Introduction

Yesterday signed the return of Advent of Code and for the first time in my life, I will try to solve the puzzles that are proposed every day.

I want to make it clear that my goal will be to propose a clear and easy to understand solution for each of the exercises. By this I mean that I will not aim for time/memory optimization. At least, not for the versions that I will document the most.

If I feel like it, I will surely recode an optimal solution.

Moreover, this year JetBrains offers to process the topics using Kotlin. Great, it's been a long time since I coded in this language!

All my solutions will be available on this GitHub repo. I also created a private leaderboard if you want to challenge me : 1663489-611b8c95.

Let's have fun!
